## Play Site Managment API

#### Example:
1. Payloads are expected in JSON format and are deserialized in snake case format, initial Play Site POST endpoint: http://localhost:8090/api/playsite
```
{
  "ball_pit_count": 2,
  "carousel_count": 1,
  "slide_count": 1,
  "double_swing_count": 1
}
 ```
#### H2 Console
- Web console can be accessed at http://localhost:8090/h2-console
- Schema and data is dropped on restart
